一、系统概述

Swaner POD Wave Viewer 是 POD（Print on Demand，按需印刷）行业的生产管理系统。主要对接领星 WMS，实现波次查询、面单打印、SKU 主数据管理和产品图片管理的全流程数字化。

访问地址：https://www.swaner.vip/

🎯 核心功能一览

模块功能入口
波次查询从 WMS 拉取波次数据、查看订单明细、打印面单首页搜索框
SKU MasterSKU 主数据管理、批量同步、概览分析📋 SKU Master 按钮
图片管理产品图片上传、缺图概览、批量删除🖼️ 图片管理 按钮
💡 数据存储说明：SKU 数据和产品图片存储在云端服务器，任何电脑打开 swaner.vip 都能访问，换电脑、清缓存不会丢失。

二、波次查询

2.1 查询波次

  1. 在首页顶部搜索框输入波次号（如 W260611CV6）
  2. 按 回车 或点击 查询 按钮
  3. 系统自动从领星 WMS 拉取该波次的所有订单数据并展示

2.2 阅读查询结果

查询结果分为两部分：

  • 顶部摘要：显示波次号、订单数、总件数
  • 明细表格：每行一个订单，每列一个 SKU 产品

2.3 SKU 类型标签

每个 SKU 单元格底部显示数量 + 彩色类型标签：

标签类型说明示例
Accessories配件支架、底座、挂钩、连接件等Round Display Stand、Rotatable Base
Standard普品标准产品标准图案产品
Customization定制品定制设计图案The Eerie Wedding、Glass House
💡 配件（Accessories）会自动排在前面，相同 SKU 的订单尽量挨在一起。

2.4 面单打印

  1. 在订单行找到 🖨️ 面单 按钮
  2. 点击后在新标签页打开面单 PDF
  3. 使用浏览器打印功能（Ctrl+P）直接打印
  4. 打印后按钮自动变为 ✅ 已打印
⚠️ 面单数据来自领星 WMS，如果显示"未找到面单"，说明 WMS 中该订单尚未生成面单，请联系领星确认。

三、SKU 管理中心

点击右上角 📋 SKU Master 按钮打开。所有 SKU 数据存储在云端，多设备共享。

3.1 Standard / Accessories 清单

  • 查看所有 Standard 和 Accessories SKU
  • 可按类型筛选、按名称搜索
  • 每次 WMS 波次查询自动收录新 SKU，维护日期为当天

3.2 Customization 清单

查看所有定制品 SKU，同样支持搜索。

3.3 📥 导入 / 批量同步

批量同步（推荐）：输入多个波次号（每行一个），点 🔄 开始同步，系统自动全部拉取并收录。

手动导入：粘贴 SKU 数据，格式为 SKU代码,类型（如 ABC001,Accessories），每行一个。

3.4 📊 概览

查看 SKU 总数、各类型数量、每日增长趋势表。

3.5 🗑️ 删除

按类型、日期、名称筛选 → 勾选 → 批量删除。删除不可恢复，请谨慎操作

四、图片管理中心

点击右上角 🖼️ 图片管理 按钮打开。

4.1 上传图片

  1. 选择对应的 Tab：Standard / Accessories 上传 或 Customization 上传
  2. 拖拽图片文件到上传区域，或点击选择文件
  3. 文件名 = SKU 代码（不含 .png/.jpg 扩展名）
  4. 上传后自动加入 SKU Master（如为新 SKU）
💡 如果上传失败，页面会显示每个失败文件的 SKU 名称 和 具体错误原因，方便排查。

4.2 📊 概览（缺图检查）

  • 显示 每个 SKU 的图片状态：✅ 有图 / ❌ 缺图
  • 可按类型筛选
  • 📋 导出缺图清单 → 下载 CSV 文件，方便发给团队找图后批量上传

4.3 🗑️ 删除图片

按类型、名称筛选 → 勾选 → 批量删除。

五、其他功能

📷 扫码

点击右上角 📷 扫码 打开摄像头，扫描波次号条形码/二维码 → 自动填入搜索框并查询。适用于生产现场快速操作。

🔄 重置打印

清除所有订单的"已打印"标记。适用于需要重新打印的场景。

📋 复制订单号

鼠标悬停在订单号上，点击出现的复制图标即可复制。

六、常见问题

Q: 查询波次提示"WMS 加载失败"？

检查网络连接是否正常。如果持续失败，可能是领星 WMS 服务异常，稍后重试。

Q: 上传图片失败怎么排查？

系统会显示每个失败文件的 SKU 和错误原因。常见原因：文件损坏（太小）、格式不支持（仅 PNG/JPG/GIF/WebP）、文件名包含特殊字符。

Q: 换电脑后 SKU 数据还在吗？

在。SKU 数据和图片都存储在云端服务器，任何电脑打开 swaner.vip 都能看到，不受本地浏览器限制。

Q: 面单按钮灰色点不了？

说明该订单在 WMS 中尚未生成面单 PDF。请联系领星确认面单是否已生成。

Q: 如何批量导入已有 SKU 的图片？

先在图片管理 → 概览 → 导出缺图清单 CSV，按清单收集图片（文件名=SKU代码），然后在图片管理上传即可。
